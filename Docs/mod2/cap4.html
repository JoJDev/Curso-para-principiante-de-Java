Ejecución de un programa de Java a través del editor de código

Antes te enseñamos como tradicionalmente se compila y ejecuta un programa hecho en Java, pero a lo largo del desarrollo de un programa, necesitaras compilar y ejecutar tu programa muchas veces para verificar que funcione adecuadamente o para hacerle pruebas a las nuevas funcionalidades que estas implementado, lo cual puede ser tedioso a lo largo del desarrollo. Por suerte, los IDE´s y los editores de código tiene (no todos) herramientas de compilación y ejecución automáticas, donde con un solo comando o clic, estos realizan la tarea de compilar y ejecutar tu código de manera automática, ayudándote a la hora que quieras probar la ejecución de tus programas de forma rápida y eficiente.

Ahora veremos como ejecutar nuestro programa HolaMundo.java en el editor de código Visual Studio Code para que te des una idea de lo fácil y sencillo que es probar tus programas en este editor de código. Para poder compilar y ejecutar código Java con Visual Studio Code, se tiene que instalar unos módulos para Java desde la pestaña de “extensiones”

Escribes “Java” en el buscador de extensiones y debes instalar el “Extension Pack for Java” desde el botón azul pequeño de “install” o entrar en esa opción e instalarlo desde el botón azul de “install”

Una vez instalados, tal vez tengas que cerrar y volver abril Visual Studio Code, pero ya podrás compilar y ejecutar tus archivos .java desde el editor de código, solo dando clic en el botón triangular de “Play” o dando en el “Run” arriba del main y se abrirá una terminal de comandos en la parte inferior con la ejecución de tu programa Java desde el propio Visual Studio Code.

También se puede compilar y ejecutar código Java desde IDEs como Eclipse o IntelliJ IDEA, pero para eso hay que crear un proyecto más formal y es un poco más complicado para los principiantes en Java, si quieres pruébalos y si le hayas el truco, puedes usarlos para este curso.
