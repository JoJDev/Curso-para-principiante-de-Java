Variables constantes

Habrá ocasiones en las que quieras tener una variable en donde una vez asignado un valor, este no cambie durante la ejecución del programa, ya sea para tener un valor constante o para que no se pueda cambiar por accidente, para eso existen las variables constantes, las cuales, una vez asignado su valor, no se pueda modificar a lo largo de la ejecución del programa, aun podrás leerlo para acceder a su valor, pero cambiarlo no podrás. Esta es su sintaxis: 

final <tipo_variable> <nombre_variable> = <valor>;

Para hacer a una variable en constante, solo hace falta poner la palabra reservada final antes de la declaración de tipo de la variable. Puedes crear una variable constante y asignarle un valor después, pero una vez que ya tenga un valor, ya no podrá modificarse.
