Instalar el JDK

Java no está preinstalado en los Sistemas Operativos desde el principio, así que hay que instalarlo primero, para ello instalamos el kit de desarrollo de Java o Java Developer Kit (JDK), que es un set de herramientas para programar con Java y debe ser instalado para desarrollar programas en este lenguaje, en él se incluyen el Java Runtime Environment (JRE), un intérprete / cargador (Java), un compilador (javac), un archivador (jar), un generador de documentación (Javadoc) y otras herramientas necesarias en el desarrollo de Java.

El Java Runtime Environment proporciona los requisitos mínimos para ejecutar una aplicación Java; consta de la máquina virtual Java (JVM), paqueterías principales y archivos de soporte. Si no quieres desarrollar programas Java, pero si poder ejecutarlos, solo instala el runtime enviroment.

Esto significa que con solo instalar el JDK, tendremos lo necesario para desarrollar en Java y para empezar la instalación, sigue los siguientes pasos:

  1. Ve a la página web oficial de Oracle (https://www.oracle.com/mx/index.html) y crea una cuenta gratuita para poder descargar el JDK

  2. Si tienes dudas de que poner en las casillas de “cargo” y “nombre de empresa” del formulario, puedes decir que eres estudiante y poner la institución en la que estudias y su dirección o si ya tienes una cuenta, puedes usar esa.

  3. Una vez creada y confirmada tu cuenta de Oracle, dirígete al siguiente enlace: https://www.oracle.com/java/technologies/downloads/ y desplázate hacia abajo hasta encontrar la sección de Java 8, Java esta hasta la versión 17, pero nosotros usaremos la versión 8, así que, selecciona la opción de Sistema Operativo que ocupes (en mi caso es Windows) y descarga el instalador según tu versión de S.O. (i586 o x64), los Windows de hoy en día suelen ser de 64 bits (x64) pero si este instalador te da error, prueba con el otro de 32 bits (i586).

  4. Una vez descargado el instalador, le das doble clic para ejecutar el instalador, las opciones que están por defecto son suficientes, así que solo hay que dale en “next” cada vez que aparezca la opción

  5. Y listo, ya tienes instalado Java 8 en tu equipo, deberías poder usar java gracias a esta instalación, pero si hay un error, revisa el siguiente capitulo que veremos a continuación.
