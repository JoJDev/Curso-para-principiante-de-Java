Instalar un editor de código

Es posible escribir y desarrollar código usando el bloc de notas tradicional del sistema, pero existen herramientas que nos ayudan a la hora de codificar los programas dándonos utilidades como resaltado de palabras clave y sintaxis, tabulación automática, compilación y ejecución del código entre muchas otras, se llaman editores de código y te recomendaremos uses alguna de las siguientes opciones.

  • Visual Studio Code

    Creado por Microsoft, es el editor de código más utilizado por los desarrolladores (no confundir con su versión de C# - Visual Studio, sin el Code), es ligero y gratuito, multiplataforma y se le puede instalar varias utilidades extra (desde su manejador de extensiones) para cualquier lenguaje de programación, incluyendo java y por eso, es el que más recomendamos usar.

    Como intalar VScode

  • Atom

    Editor de código moderno y ligero para escribir código de varios lenguajes y una gran alternativa de Visual Studio Code, si no te gusto el anterior editor de código, usa este, es practico y simple para los que apenas comienzan a programar, solo abres un documento nuevo, específicas el nombre y la extensión y listo, a programar.

    Como instalar Atom

  • Notepad ++

    Algo antiguo, pero igual de útil es el editor de código Notepad ++, que maneja varios lenguajes de programas, color de sintaxis y utilidades básicas para desarrollar código en él, es portable, sencillo y gratuito, una opción más si Visual Studio Code no te convenció.

    Como instalar Notepad ++

  • IntelliJ IDEA

    Más que un editor de código, IntelliJ IDEA es un entorno de desarrollo integrado (IDE) que es un software para el diseño de aplicaciones que combina herramientas comunes para desarrolladores en una sola interfaz de usuario gráfica, y este es especializado en Java, Aunque puede ser complejo de usar, tiene todo lo que un desarrollador en Java necesita, pero no se recomienda usarlo si eres principiante ya que es complicado de emplear y no está en español, lo que dificulta su uso en recién entrados al desarrollo de software.

    Como instalar IntelliJ IDEA

  • Eclipe

    Eclipse también es un IDE muy bueno para Java SE, no tan liviano como un editor de código, pero muy útil a la hora de desarrollar aplicaciones en java y en los otros lenguajes que soporta, es un poco antiguo, pero adecuado para los que recién empiezan a desarrollar programas informáticos y no tiene nada que envidiarle a otros IDEs como InteillJ IDEA.

    Como instalar Eclipe

Cualquiera de estas opciones está bien para este curso, pero recomendamos usar Visual Studio Code (e instalarle las extensiones de Java) porque ese es el editor de código que se usó en la creación de este curso y es la herramienta más accesible para los nuevos a la hora de programar.
