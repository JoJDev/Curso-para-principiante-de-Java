Crear las variables de entorno

Si todo salió bien, ya podrás usar Java sin problemas, para comprobar lo, busca en el buscador de programas un programa llamado “CMD” o símbolo del sistema que sirve para ejecutar comandos en la PC y abre lo, la tienen todos los Windows (o una “terminal” para otros S.O.), aquí escribimos el comando “java -version” y si te muestra la versión de java que instalaste, todo está correctamente instalado.

Pero si no da con el resultado anterior, tocara moverle un poco a tu PC para poder ejecutar Java y eso lo hacemos en las variables de entorno. Las variables de entorno son para guardar y consultar información que forma parte del Sistema Operativo en el que se ejecuta un programa y son necesarias para el funcionamiento de éstos. Por ejemplo, un programa o proceso en ejecución puede consultar el valor de la variable de entorno TEMP para descubrir una ubicación adecuada para almacenar archivos temporales, o la variable HOME o USERPROFILE para encontrar la estructura de directorios propiedad del usuario que ejecuta el proceso.

Y hablamos de variables de entorno porque Java requiere que indiquemos que indiquemos en la variable de entorno PATH la ruta de sus ejecutables para así poder usar sus comandos y herramientas de desarrollo, y para ello, sigue los siguientes pasos:

  1. Abre un explorador de archivos y haz clic derecho en el icono de “este equipo” y selecciona la opción de “propiedades”.

  2. Aparecerá una ventana con las características de tu sistema, vas a hacer clic en el enlace “configuración avanzada del sistema” en la parte izquierda (en Windows 10 se localiza a la derecha, en Windows 8 a la izquierda) para abrir la sección de “propiedades del sistema”

  3. Aparecerá otra ventana un poco más pequeña, le damos clic en la opción “variables de entorno” en la parte de abajo

  4. Entonces se abrirá la ventana de las variables de entorno, estará dividida en dos secciones, variables de usuario (arriba) y variables de sistema (abajo), ahora hacemos doble clic a la variable PATH del sistema (abajo) para poder agregar la ruta del directorio de Java.

  5. Para ello, en la parte de “valor de la variable” hay que agregar la dirección en tu disco duro de los binarios de Java, generalmente se encuentra en C:\Program Files\Java\\bin, pero asegúrate visitando el enlace, si todo está en orden, agrega (no borres nada, agrega) el siguiente texto (sin las comillas) al final de la cadena de texto “;C:\Program Files\Java\jdk1.8.0_321\bin”, esto es la dirección en el PC de java que comprobaste (si tienes Windows 10, ponlo hasta debajo de la lista). Pon en “aceptar”

  6. Y eso sería todo, vuelve a comprobar si Java esta bien instalado con el CMD y si aun hay error, revisa que hayas agregado la ruta adecuadamente en la variable de entorno indicada.
