Ejecución de un programa de Java a través de CMD

Ya tenemos nuestro código, ahora hay que compilarlo y ejecutarlo en nuestra computadora, para ello deberemos de ir a donde se localiza el archivo Java que escribimos desde el CMD o símbolo del sistema y tenemos dos formas de hacer eso:

  1. Copiamos la dirección desde el explorador de carpetas y pegamos esa dirección junto al comando cd (ejemplo: cd <ruta/”de tu”/carpeta>), si tienen espacios, pon la frase entre comillas dobles, como se muestra en la siguiente imagen:

  2. Desde el explorador de carpetas y estando en la carpeta donde guardaste tu archivo java, en la barra de ruta, escribe “cmd” (sin las comillas dobles) y da un enter para abrir el símbolo del sistemas (CMD) en la dirección donde estabas, como se ve en la imagen:

Una vez dentro, hay que compilar el código java, esto se hace desde el cmd con el comando “javac HolaMundo.java” y dando ENTER empieza la compilación del archivo que hayamos especificado, en este caso HolaMundo.java, una vez terminado el proceso y si no tiene ningún error de sintaxis (si ese fuera el caso como en la imagen de abajo, revisa que no haya errores de ortografía ni que falte alguna letra o símbolo y revisa que respetes las minúsculas o mayúsculas, ya que java es muy estricto en ese aspecto), tendrás un nuevo archivo con la extensión .class que es el programa listo para ejecutarse por Java .

Para ejecutar tu programa, una vez compilado, en el CMD usa el comando “java HolaMundo” y veras como tu mensaje ha sido escrito en pantalla.

¡Felicidades! Has escrito, compilado y ejecutado tu primer programa en Java. Ahora prueba en cambiar el mensaje que muestra en pantalla y no te preocupes si no lo logras a la primera, tanto en la programación como en la vida, de los errores se aprende.
