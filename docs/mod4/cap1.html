Introducción: ¿Qué son las cadenas de texto?

Las cadenas de texto son un tipo especial de dato que sirven para guardar texto y, al contrario de un carácter, este es un tipo especial (no-primitivo) porque es prácticamente un conjunto de caracteres consecutivos en cadena, por eso se le llaman "cadena de texto" o String (cadena en inglés), pueden parecer un tipo de dato primitivo más pero tiene que ver más con los vectores; que son variables que pueden guardan otras variables en su interior, como una variable de variables (y que veremos más a futuro) que con los propios caracteres.

Esto es así para poder manipular de varias y diversas maneras un dato tipo String (cadena de texto), poderle agregarle más texto, sustraer una sub-cadena de texto de la propia cadena, buscar un conjunto de caracteres o sub-cadenas, entre muchas otras operaciones.

Es muy común usar texto en la programación, tanto para dar mensajes como para recibir información del usuario o de cualquier otro medio (generalmente se recibe en forma de texto), por eso tiene un módulo entero donde se ahondará en cómo crearlas y manipularlas, las operaciones que puedes realizar con ellas, calcular su longitud y otras funciones y comandos para sacarles el máximo provecho.

Un ejemplo de uso de las cadenas de texto es el propio CMD, que solo se comunica con texto, y si pide datos, se la daremos en forma de texto (aun si le pasamos números, este los tomara como texto), como se ve en la imagen de abajo que muestra un programa que veremos en posteriores módulos pero muestra claramente el uso de cadenas de texto en Java
