  1. Crea un programa que cree y muestre una variable de cada tipo con las siguientes características (tú puedes elegir el nombre de las variables):

    • Una variable tipo int que contenga el valor 1,478,963,250
    • Una variable tipo long que contenga el valor del int anterior, pero x2
    • Una variable tipo float que contenga el valor 9.9
    • Una variable tipo double que contenga el valor PI con 4 dígitos después del punto
    • Una variable tipo booleana que contenga el valor de falso
