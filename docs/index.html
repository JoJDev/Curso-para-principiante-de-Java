Bienvenido a este curso de programación para principiantes usando Java

Este curso esta pensado para los estudiantes de nuevo ingreso de las carerras de Licenciatura en Ingeniería en computación (ICO) y Licenciatura de Informática administrativa (LIA) del Centro Universitario UAEM Ecatepec para introducirse a la programación con Java, uno de los lenguajes de programación más demandados en el mercado y que ayudara en los futuros estudios de su carrera.

Para tomar este curso no se requiere algun conocimineto en especifico, solo los conocimientos basicos para manejar una computadora, por lo que cualquiera pueden tomar este curso,

En este curso aprenderas:

  • ¿Que es la programación? y un resumen de la histaria de la programación
  • Instalar, crear y ejecutar programas Java
  • Crear y manejar información a traves de variables
  • Que tipos de datos existen en Java, como usarlos y convertirlos de un tipo a otro
  • Mostrar y pedir informacion al usuario para trabajar con ella
  • Operadores y expresiones que se usan en Java
  • Estruturas de decisión e iteración para manejar el flujo de ejecucion de tu programa
  • Usar vectores y matrices en java
  • Programacion Orientada a Objetos básica
  • Manejo de errores y excepciones

El curso esta dividido en modulos, cada uno con sus temas, ejericios y pregutas para que practiques, ademas de un diccionario con las palabras tecnicas que se usan y que puedes consultar cada vez que lo requieras. El temario esta en la parte derecha o si esta viendo el curso en un dispositivo movil, se mostrara un boton para desplegar el temario en pantalla (al igual que el diccionario)

Para efectos educativos, podras encontrar a lo largo de los modulos una aplicacion enbebida llamada paiza.io que nos ayuda a ejecutar directamente los programas Java que estamos usado de ejemplo, la primera seccion es para escribir tu codigo y en la pequeña seccion de abajo es el resultado.
