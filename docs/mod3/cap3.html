Crear variables

Para crear una variable debes indicar el tipo de dato que guardara y como se llamara, para poder referirnos a ella más adelante en el programa, por lo que podemos decir que, por ejemplo, “la variable es de tipo ”, ya que, una vez creada, una variable no puede cambiar el tipo de dato que guarda. Para crear una variable en Java, sigue la siguiente estructura y no olvides el punto y coma ( ; ): 

<tipo_variable> <nombre_de_la_variable>;

Esto crea la variable, pero no contiene ningún valor, para utilizarlas, primero debemos darle un valor, esto lo hacemos con el símbolo de igual ( = ), sigue la siguiente sintaxis: 

<nombre_variable> = <valor>;

Veamos en el programa 2 como crear, asignarle y posteriormente visualizar un valor de una variable tipo int.

Como vemos, en la línea 4 declaramos una variable de tipo int con el nombre de numero y después en la línea 5 estamos diciendo que la variable numero que creamos sea igual a 4, es decir, le estamos asignando un valor a la variable para que lo guarde y en la línea 7, estamos pidiendo que muestre el valor con la función “System.out.println( );” (numero no está entre comillas dobles aquí porque es una variable de tipo int, no una cadena de texto) y efectivamente, el valor de nuestra variable se muestra en consola.

También podemos asignar el valor de una variable en la misma línea en la creamos y vemos que sigue siendo lo mismo

Aquí el tipo de variable no afecta, esta es una forma rápida de crear y asignar variables, de hecho, la única razón para solo crear una variable sin asígnale seguidamente de un valor es por qué le vas a dar un valor después durante la ejecución del programa.
